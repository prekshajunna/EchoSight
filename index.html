1. Introduction

Blind and visually impaired individuals face challenges in navigating their environment. Our system provides AI-powered assistance through real-time object detection.

2. Project Description

This project helps visually impaired users by identifying nearby objects using smartphone cameras.

3. Technologies Used

• AI & ML for object detection.
• TensorFlow.js for in-browser deep learning.
• COCO-SSD for real-time object detection.

4. Features

✅ Detects objects in real-time.
✅ Provides vibration feedback for close objects.
✅ Announces object positions.

5. System Implementation

5.1 Object Detection Module

The COCO-SSD model identifies objects and determines their location using TensorFlow.js. Distance estimation is used to provide appropriate feedback.

5.2 Sensory Feedback Mechanisms

• Audio Feedback: Announces detected objects through text-to-speech.
• Haptic Feedback: Vibrates when objects are too close.
• Visual Effects: Highlights objects for those with partial vision.

7. Conclusion

The AI-Based Blind Assist System combines AI, computer vision, and smartphone technology to empower visually impaired individuals with enhanced mobility and independence.
